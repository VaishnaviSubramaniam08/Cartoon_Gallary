# 🎨 Cartoon Gallery

A collection of fun, minimalist cartoon faces created entirely using **HTML** and **CSS**.

## 🚀 Features

- **Pure CSS Art**: No external images or SVGs used. Everything is rendered using CSS shapes and properties.
- **Responsive Layout**: Uses Flexbox for centering and positioning.
- **Minimalist Design**: Clean and simple visual style.

## 🖼️ Gallery Items

### 1. Catface (`.\catface.html`)
A cute orange cat face featuring:
- Pointy ears
- Round eyes
- A pink nose
- A smiling mouth
- *Motto: "Eat. Sleep. Meow. Repeat."*

### 2. Smiley Face (`.\smiley.html`)
A classic cheerful yellow smiley face featuring:
- Large round eyes
- A wide, friendly smile
- *Motto: "Smile — it’s contagious!"*

## 🛠️ How to Use

Simply open the HTML files in any modern web browser to view the art:


Alternatively, just double-click the files in your file explorer.

## 🎨 Customization

You can easily customize the colors and sizes by modifying the `<style>` block in each HTML file.
- Change `background-color` to give the characters a new look.
- Adjust `width` and `height` to resize different features.
